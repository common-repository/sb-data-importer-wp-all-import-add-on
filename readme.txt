SB Data Importer allows you to import ads, listing in minutes if you're using Adforest.

== Description ==
SB Data Importer is a lightweight plugin which allows you to import ads, listing in minutes by using <a href="http://www.wpallimport.com/" target="_blank">WP All Import</a>. It's a simple but advanced plugin with the most pivotal functionalities, created to save you time.

A plugin that makes it easy to import XML and CSV files into <a href="https://themeforest.net/item/adforest-classified-wordpress-theme/19481695" target="_blank">Classified WordPress theme</a> as well as <a href="https://codecanyon.net/item/adforest-classified-native-android-app/20963101" target="_blank">classified ads app</a>. It will import your existing ads, and listing into our themes to make complicated import tasks much faster and easy.

There are no extra requirements that the data in your file must be laid out in a certain way. SB Data Importer really can import any XML or CSV file.

So let's take a closer look at the features of SB Data Importer.
*	Easily integrated.
*	A fast and cost-effective plugin.
*	No code necessary.
*	Import listing details with a glance.

Supported themes are:
- AdForest WordPress Theme
- Carspot WOrdPress Theme
- DWT Directory Listing Theme
- Nokri Job Board WOrdPress Theme

== Installation ==
1. Upload "adforest-classified-theme-add-on.zip" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==
= Can it be use indiviually or must be with ScriptsBundle's themes?=
It must be use with Adforest and Opportunities themes and will be added support for more theme in near future.

= Can it be use without WP all import plugin? =
It's an add-on of WP all import so it can't be use without that.
